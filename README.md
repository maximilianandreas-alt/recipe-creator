## Setup
1. `npm install`
2. Create a `.env` file with your **Neon Database** connection string (FR006).
3. `npm run dev`

## Git Workflow (Mandatory)
- Never push to `main`.
- Open a PR for every feature (e.g., `feature/neon-integration`).
- Use TypeScript for all new components.
